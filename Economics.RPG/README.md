# Economics.RPG 升级职业插件

- 作者: 少司命
- 出处: 无
- RPG 升级插件

> [!NOTE]  
> 需要安装前置插件：EconomicsAPI (本仓库)


## 指令

| 语法           |        权限         |   说明   |
| -------------- | :-----------------: | :------: |
| /rank [职业名] | economics.rpg.rank  |   升级   |
| /重置等级      | economics.rpg.reset | 重置职业 |
| /level reset   | economics.rpg.admin | 重置     |
