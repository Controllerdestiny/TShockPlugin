# SimultaneousUseFix 解决卡双锤卡星旋机枪之类的问题

- 作者: 肝帝熙恩，恋恋
- 出处: 无
- 解决卡双锤卡星旋机枪之类的问题
- 因为一些检测原因，最后一个桶会被判定，所以加了免检测
> [!NOTE]  
> 需要前置：https://github.com/sgkoishi/yaaiomni/releases
